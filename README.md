## Kyber NTT 연산 파이썬 구현 예제
이 리포지토리는 PQC 표준 알고리즘인 Crystal-Kyber 논문을 기반으로, 핵심 연산(NTT, 다항식 연산)을 파이썬으로 구현한 학습용 코드입니다.

**주요 구현 내용**
- NTT (Number-Theoretic Transform)
- 역NTT (Inverse NTT)
- NTT 기반 다항식 덧셈 및 곱셈
  

## ⚠️ 주의 
이 코드는 논문의 개념을 이해하고 학습하기 위해 작은 값(예: mod 13)을 사용한 예제입니다. 절대로 실제 보안 환경에서 사용해서는 안 됩니다.
